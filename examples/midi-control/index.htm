Audiolet - MIDI Control Example

This demonstrates a simple MIDI chain where a keyboard controls an instrument. Click on the document and use your a, w, s, e, d, r, f, g, y, h, u, j, k, o, l, and p keys to make noises.

js/midi-control.js

          
var audiolet = new Audiolet(),
    keyboard = new MidiKeyboard(audiolet),
    instrument = new Instrument(audiolet);

keyboard.connect(instrument);
instrument.connect(audiolet.output);
